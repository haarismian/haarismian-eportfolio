Reflections of the activity and course learning objectives

This Python program requires the use of many skills, including problem-solving and IT and digital skills. As the program must retain employee details and allow for annual leave requests, it requires the programmer to think critically and analyze the best way to organize and store this data. Time management is also important as the programmer needs to allocate enough time to create a working program while meeting any deadlines. Additionally, communication and literacy skills come into play as the programmer needs to effectively document their code and any issues they encounter. Numeracy skills may also be necessary when handling any calculations or mathematical operations within the program. Finally, critical reflection is important as the programmer should evaluate their code and any problems they encountered to improve their skills for future programming projects.
