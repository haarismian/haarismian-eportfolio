Object Oriented Programming

Learning Objectives

  • Develop an understanding of object oriented programming using Python and its major features: abstraction, encapsulation, inheritance, and polymorphism.
  • Gain experience in designing object oriented programs using the Unified Modelling Language (UML).
  • Understand the use of data structures to support the retention of data in the most effective way for the data in question.
  • Learn how to search data in the most efficient manner.
  • Understand the role played by design patterns in the creation of software.
  • Develop the ability to develop an object oriented program using Python.
  • Be able to prepare UML models to support the object oriented design process.
  • Apply data structures to support the storage of data in a variety of ways, implement data search algorithms to process stored data in the most efficient approaches, and use design patterns to organize code contributing to an individual system.
