Unit 1

Learning Objectives

  • Analyze the concept of object-oriented programming.
  • Examine the syntax utilized for defining Python classes.
  • Investigate how to establish various data types in Python.
  • By the end of this unit, learners will be able to:
  • Explain the key characteristics of an object-oriented program, including abstract classes, inheritance, encapsulation, and polymorphism.
  • Create a Python class.
  • Assign various data types to variables in a Python program and customize the access modifier for each.
  • This unit serves as a foundation for more advanced Python programming concepts that will be covered in subsequent weeks.

Reflections of the activity and course learning objectives

I found Week 1 to be an informative introduction to Python programming. The unit provided an overview of the evolution of programming languages towards object orientation, which is a key concept in modern software development. The unit's learning outcomes were well-structured and provided a clear roadmap for developing proficiency in Python programming. Specifically, I appreciated the emphasis on the major features of an object-oriented program, such as abstract classes, inheritance, encapsulation, and polymorphism.

The formative activities were helpful in allowing me to apply the concepts learned in the unit, including defining a Python class and adapting the access modifier for different data types. The Codio Activity, in particular, was useful in providing hands-on experience with defining classes and objects in Python. I also found the Collaborative Discussion and e-Portfolio Activities to be engaging opportunities for discussing and sharing my experiences with fellow students.

Overall, Week 1 provided a solid foundation for future learning in the course. I appreciated the clear objectives and well-designed activities, which allowed me to develop a practical understanding of Python programming's fundamental building blocks. I am excited to continue my learning and see how these concepts are applied in more advanced topics in the weeks to come.
