Unit 5

Learning Objectives

  • Explore the range of questions to ask when developing a test plan.
  • Become familiar with two industry software testing standards.
  • Examine a range of testing techniques and appreciate the relevance of each in different testing scenarios.
  • Recognise the tools and frameworks which are available to support and automate the Python testing process.
  • Describe the key terms which are associated with testing software for security.

Reflections of the activity and course learning objectives

I'm very excited to better understand the essential questions to ask when developing a test plan, familiarizing myself with industry software testing standards, and delving into the diversity of testing techniques.This knowledge is crucial in effectively identifying vulnerabilities in software, hence preventing potential breaches. Furthermore, recognizing the array of tools available to automate the Python testing process will undoubtedly increase my efficiency and productivity as a software engineer, allowing me to allocate more time to other critical aspects of software development, especially as test driven development rises in popularity

I'm also excited to learn what happens to software that is breached despite being robustly tested
