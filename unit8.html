Unit 8

Learning Objectives

  • Understand and describe the set operations of union, intersection, difference, and symmetric difference.
  • Identify and explain the scenarios in which each set operation is applicable to the driverless car scenario.
  • Gain an appreciation of the linear data search technique and its application in data processing.
  • Implement a linear data search technique to process stored data.
  • Apply the knowledge of set operations and linear data search technique to implement the key features required in the summative code assessment, which include data structures and data search algorithms.

Reflections of the activity and course learning objectives

Problem-solving: This task requires identifying a recursive algorithm to solve the problem of raising the base integer to the exponent integer.

IT and Digital Literacy: This task involves implementing the recursive function in Python using proper syntax and data types.

Critical Reflection: Upon completion of the task, the student may reflect on the process of designing and implementing the recursive algorithm and evaluate its effectiveness in solving the problem.
